Formatações

Negrito

Existem duas maneiras de se colocar um texto em negrito. Usando a tag b ou usando a tag strong

Itálico

Assim como o negrito, existem duas maneiras de colocar um texto em itálico. Usando a tag i ou usando a tag em

Marcação

É possível marcar um texto dessa forma.

Texto riscado

É possível riscar um texto dessa forma.

Texto sublinhado

É possível sublinhar um texto de duas formas. Usando a tag u ou usando a tag ins.

Texto sobrescrito

É possível sobreescrever um texto da seguinte forma:
a2 = b2 + c2

Texto subscrito

É possível subscrever um texto da seguinte forma:
H2O.
